# ADR-0001 · UI 기술스택

> 상태: ✅ 결정됨
> 일자: 2026-06-22
> 관련: [`docs/01.기획/프로젝트-브리프.md`](../01.기획/프로젝트-브리프.md), [`docs/01.기획/리서치.md`](../01.기획/리서치.md), [ADR-0002](./ADR-0002-적응형-표현-전략.md)

## 1. 맥락

이 프로젝트의 본질은 **하나의 코드베이스로 데스크톱·모바일 웹에서 각각 자연스러운 경험을 주는 반응형 UX 전략**을 시연하는 것이다. UI 토대는 그 시연의 그릇이되, 모바일에서 앱처럼 느껴지는 경험을 직접 통제할 수 있어야 한다.

평가 기준(우선순):
1. **디자인 소유 / 제어권** — 표현을 우리가 완전히 통제할 수 있는가
2. **모바일 UX 자력** — 적응형 시트·제스처·접근성을 어디까지 받쳐주는가
3. **CSR 정합** — 클라이언트 전용 환경에 맞는가(SSR 가드 부담 없음)
4. **번들 가벼움**

판단 기준은 **"외형은 소유하되, 복잡한 동작 로직은 검증된 것을 활용한다"** 이다([리서치 §2](../01.기획/리서치.md)).

## 2. 검토한 대안

| 선택지 | 디자인 소유 | 모바일 UX | CSR 정합 | 비고 |
|--------|:---:|:---:|:---:|------|
| 풀 컴포넌트 키트 (Material/PrimeNG) | ✕ | △ | ○ | 디자인 언어가 따라옴 |
| Ionic | ✕ | ◎ | ○ | Shadow DOM이 Tailwind를 가로막음 |
| spartan/ui | ○ | ○ | ○ | 같은 토대(CDK+Tailwind), 참고 구현으로 활용 |
| **Angular CDK + Tailwind v4 + 직접 구현** ✅ | ◎ | ○ | ◎ | 동작·접근성만 빌리고 마크업은 light DOM |

## 3. 결정

**Angular CDK(헤드리스) + Tailwind CSS v4 + 직접 구현한 모바일 레이어**를 채택한다.

- **CDK**: Overlay(위치·백드롭·포커스 트랩), A11y, Scrolling(가상 스크롤), DragDrop, BreakpointObserver. 마크업이 전부 light DOM에 남아 Tailwind가 모든 요소에 적용된다.
- **Tailwind v4**: `@theme` 토큰 기반으로 디자인을 직접 조립.
- **직접 구현**: safe-area, 스크롤 물성처럼 단순·선언적인 영역.
- **검증된 라이브러리(후보)**: 끌어서 닫는 시트 등 고수준 제스처는 GSAP Draggable/InertiaPlugin·Interact.js·@use-gesture 중 첫 시트 구현 때 선정.
- **spartan/ui**: 같은 토대를 쓰므로 직접 만들기 전 참고 구현으로 둔다.
- **적응형 표현**: CDK Overlay + BreakpointObserver로 공용 `AdaptiveSheet`/`AdaptiveDialog`를 캡슐화한다(→ [ADR-0002](./ADR-0002-적응형-표현-전략.md)).

## 4. 주의사항 (CSR 전제)

- 이 프로젝트는 CSR 전용이라 SSR 하이드레이션 불일치·플랫폼 가드 부담이 없다. (자매 프로젝트 `angular-ssr-design`은 spartan/ui를 골랐지만, 그건 SSR 정합이 1순위였기 때문 — 우리는 기준이 달라 결론도 다르다.)
- 다만 `BreakpointObserver`·DOM 측정은 여전히 브라우저 런타임 의존이므로, 초기 분기는 **CSS 우선**으로 처리해 깜빡임을 줄인다.

## 5. 채택하지 않은 것

- **풀 컴포넌트 키트**: 디자인 언어가 함께 따라와 "디자인 소유" 목표와 충돌.
- **Ionic**: 모바일에 강하지만 Shadow DOM 캡슐화로 외부 Tailwind가 내부에 닿지 않아, 컴포넌트 내부까지 통제하려는 의도와 충돌.
- **spartan/ui 전면 채택**: 디자인이 어느 정도 입혀져 있어 "완전 소유"와 한 발 떨어진다. 참고용으로만.

## 6. 결과

- 👍 디자인 완전 소유 + 검증된 접근성을 동시에 얻는다.
- 👍 Tailwind가 모든 요소에 적용, 데모 코드가 투명하다.
- 👎 적응형 시트·제스처를 직접 조립해야 한다(비용은 ADR-0002·0008에서 흡수).
