# 설계 (Design Decisions)

이 폴더는 프로젝트의 **기술·UX 결정사항**을 ADR(Architecture Decision Record) 형식으로 기록한다.
도메인/기능 기획은 [`docs/01.기획/`](../01.기획/)에서 다룬다.

이 프로젝트의 관심사는 **CSR + 반응형 사용자 경험**이다(SSR은 자매 프로젝트 `angular-ssr-design`의 주제).
따라서 ADR도 렌더링 전략보다 **적응형 표현·인터랙션·접근성**에 무게를 둔다.

1. **shared/ui 순수 UI** — CDK Overlay 기반 공용 `AdaptiveSheet`/`AdaptiveDialog`, 오버레이 닫기 디렉티브 ([0001](./ADR-0001-ui-기술스택.md), [0002](./ADR-0002-적응형-표현-전략.md), [0003](./ADR-0003-뒤로가기-오버레이-닫기.md))
